Como instalar Docker en Windows y Mac

En este mini tutorial te mostrare como instalarlo en Windows y Mac

Instalación para Windows

Para instalar Docker en Windows se debe ingresar a: https://docs.docker.com/docker-for-windows

Pre requisitos:

  • Docker para Windows requiere Windows 10 Pro de 64 bits con Hyper-V disponible. Consulte Qué debe saber antes de realizar la instalación para obtener una lista completa de los requisitos previos.

Descargamos el ejecutable dando click en Get Docker for Windows (Stable)

  1. Haga doble clic en Docker para Windows Installer.exe para ejecutar el instalador
  2. Siga el asistente de instalación para aceptar la licencia, autorizar el instalador y continuar con la instalación.
  3. Haga clic en Finalizar en el cuadro de diálogo de con guración completa para abrir Docker

Verifica que este instalado, luego inicia Docker, cuando en la barra de estado aparece el icono de Docker vemos que está funcionando, ya se puede acceder desde cualquier ventana de terminal.

Instalación para MAC

Pre requisitos:

  • Docker para Mac requiere OS X El Capitan 10.11 o una versión más nueva de macOS ejecutándose en una Mac 2010 o posterior, con el soporte de hardware de Intel para la virtualización MMU. La aplicación se ejecutará en 10.10.3 Yosemite, pero con soporte limitado. Consulte Qué debe saber antes de realizar la instalación para obtener una explicación completa y una lista de requisitos previos con al menos 4 GB de RAM

Para instalar Docker en MAC se debe ingresar a: https://docs.docker.com/docker-for-mac

Para descargar el instalador se da click en Get Docker for Mac (Stable)

  1. Una vez tengamos el instalador en nuestro equipo, haga doble clic en Docker.dmg para abrir el instalador, luego arrastre Docker a la carpeta Aplicaciones.
  2. Con lo anterior Docker esta instalado en nuestro equipo. Haga doble clic en Docker.app en la carpeta de aplicaciones para iniciar Docker. (En el siguiente ejemplo, la carpeta de aplicaciones está en el modo de vista de “cuadrícula”).
  3. El icono en la barra de estado superior indica que Docker se está ejecutando y se puede acceder desde una terminal.

Con lo anterior ya tendremos Docker en nuestra maquina y estaremos listos para utilizarlo.
